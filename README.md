twitter-stream-pipeline
===========

Easily collect tweets off of the Twitter streaming API and store them in SQLite3. 

Setup
-----
Create a tokens.py file with your Twitter API tokens
```python
CONSUMER_KEY = ''
CONSUMER_SECRET = ''
ACCESS_TOKEN = ''
ACCESS_TOKEN_SECRET = ''
```
